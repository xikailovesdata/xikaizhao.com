Installation

Follow the steps below to get started with your Site Template:

  1. Open the Package/HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • HTML/css - Extra Stylesheets Folder
    • HTML/fonts - Custom Fonts
    • HTML/img - Images Folder
    • HTML/js - Javacripts Folder
    • HTML/style.css - Main Stylesheet File
    • HTML/index.html - Index File
    • Ultra comes with 2 home version. If you want to use another home version, just rename that home version. (Ex - If you want to use "index-vertical.html", just rename it index.html. Make sure you lst rename default index.html file)
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Changing Fonts

Change your fonts on the fly as we have included a fonts file to manage the custom fonts you include with ease. By default, Ultra uses 2 fonts namely: Dosis & Raleway from the Google Fonts Library. You can find the linking to the font files in the head tag of all the .html files.

<link href='https://fonts.googleapis.com/css?family=Dosis:400,500,600,700' type='text/css'>
<link href='https://fonts.googleapis.com/css?family=Raleway:300,400,500,700' rel='stylesheet' type='text/css'>


In order to change the fonts, you will need to edit the above links with your custom font if you plan to use a google font or remove it complete if you plan to use a self hosted font. Here is an example for using Self Hosted Fonts.
